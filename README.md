# 🏴 Projeto Luffy

Um site fictício dedicado a **Monkey D. Luffy**, o protagonista do anime e mangá **One Piece**, criado por Eiichiro Oda.

## 📋 Descrição

Este projeto apresenta uma página web temática sobre Luffy e seu sonho de se tornar o **Rei dos Piratas**. O site inclui seções interativas com fotos, informações sobre o personagem e sua jornada épica.

## 🎨 Características

- **Seção Home**: Bem-vindo à aventura de Luffy
- **Galeria de Fotos**: Imagens temáticas que formam a frase icônica "Eu vou ser o Rei dos Piratas"
- **Seção Sobre**: Informações detalhadas sobre Monkey D. Luffy e sua importância na história
- **Design Responsivo**: Otimizado para diferentes tamanhos de tela
- **Tipografia Customizada**: Utiliza a fonte "Space Mono" do Google Fonts

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica
- **CSS3**: Estilização (arquivo `style.css`)
- **Google Fonts**: Tipografia personalizada
- **Favicon**: Ícone personalizado da bandeira de Luffy

## 📂 Estrutura do Projeto

```
site_ficticio/
├── index.html          # Página principal
├── style.css           # Estilos do site
└── Luffys_flag_icon-icons.com_128.ico  # Favicon
```

## 🌐 Navegação

- **Home**: Seção inicial com chamada à ação
- **Fotos**: Galeria interativa com imagens de Luffy
- **Sobre**: Informações sobre o personagem

## 💡 Detalhes

O site utiliza:
- Links de navegação suave entre seções
- Efeitos hover nas imagens da galeria
- Estrutura semântica HTML para melhor acessibilidade
